⚠️ Observação: Apesar de parecer uma tabela, essa estrutura usa apenas <div>s. Se você copiar e colar diretamente no Excel, as informações ficarão desorganizadas.

Uma das formas de contorno é copiar primeiro para o notepad e depois, a partir dali, copiar e colar no Excel. Mesmo assim, esse método alternativo não funciona em todos os casos. O melhor é que a tabela já esteja formatada corretamente no original (com a tag table e demais tags específicas de tabelas.
